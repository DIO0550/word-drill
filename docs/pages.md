# word-drill ページ構成

## ルーティング

```
/                         ホーム（カテゴリ一覧）
/quiz/:category           クイズ設定画面
/quiz/:category/play      クイズ実行画面
/stats                    学習統計
```

## カテゴリ一覧

| category | 名前 | 説明 |
|----------|------|------|
| `general` | 一般英単語 | 日常・ビジネス英語 |
| `general-toeic` | TOEIC向け | TOEIC頻出単語 |
| `tech-programming` | プログラミング | 開発用語全般 |
| `tech-infra` | インフラ/クラウド | AWS, Docker, K8sなど |
| `tech-security` | セキュリティ | 脆弱性、認証など |
| `tech-docs` | 論文/ドキュメント | 技術文書で使う表現 |

## 各ページ詳細

### / (ホーム)

- カテゴリ一覧をカード形式で表示
- 各カテゴリの登録単語数を表示
- 統計ページへのリンク

### /quiz/:category (クイズ設定)

- カテゴリ名の表示
- 設定項目
  - 問題数: 10 / 20 / 全部
  - モード: 意味を当てる / 単語を当てる / ランダム
  - シャッフル: ON / OFF
- 「開始」ボタンで /quiz/:category/play へ

### /quiz/:category/play (クイズ実行)

- 進捗表示 (3 / 10)
- 問題カード
  - 単語 or 意味の表示
  - 例文（あれば）
  - 4択ボタン
- 回答後
  - 正解/不正解の表示
  - 正しい意味の表示
  - 「次へ」ボタン
- 終了時
  - 結果サマリー（正答数、正答率）
  - 「もう一度」「カテゴリに戻る」「ホームに戻る」

### /stats (学習統計)

- 全体統計
  - 総回答数
  - 全体正答率
  - 学習日数 / 連続学習日数
- カテゴリ別正答率
  - 棒グラフ or プログレスバー
- 苦手な単語リスト
  - 正答率50%以下の単語
  - 各単語の正答率、回答回数
- 履歴リセットボタン
